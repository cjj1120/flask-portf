Collaborative WEB3 DEFI ANALYTICS PROJECT

This is a hobby project completed by a group of Web3 and Defi enthusiasts, my friend from Singapore and me. It took about a month and we are happy with the learning journey as well as the outcome!

This project aims to provide an overview of the activities and cashflow in the decentralized finance sector. We have built a complete ETL (Extract, Transform, Load) pipeline and deployed to cloud. The data ingestion script (Extract) and data transformation script (Transform) were containerized with Docker and deployed to Google Cloud Run. The transformed data is then loaded (Load) to Google Big query data warehouse. The final dashboard is built with Metabase and deployed on Heroku.

The pipeline has been revised several times to improve performance and save cost, at the beginning we used Google Postgres SQL and later switched to Big Query which saves much more cost.

Analytics was done in SQL on the Metabase platform, the dashboard has a custom ranking feature that provides a better approach to observe the protocols performance. The final dashboard is a bit laggy since we are running on free tier on Heroku, and we didn't set a high computation configuration on GCP to save cost.

It was a long and fruitful journey, and I'm glad to work on project that I'm interested in (Defi & Web3)!
